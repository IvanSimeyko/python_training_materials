Типизация динамическая - тип связыватеся с объектом. Переменные хранят ссылки на объекты в виде значений. Переменные на которые нет ссылок автоматически освовождаются из опереативной памяти системой "сборк мусора". 

Для двух разных объектов идентификатор объекта всегда разный. И наоборот

b = 1
print(id(a) == id(b))     # True
c = 2
print(id(a) == id(c))    # False

Строгая типизация - не производится автоматических приведений типов в операциях (кроме очеивидных int=float)
