С помощью классов можно создать свой тип объекта.

Тело класса исполняется в момент определиния самого класса (в ф-х наоборот).
За классом закрепляется собственный namespace.

Конструктор (__init__(self, ...)) - определяет некоторые значения атрибутов по умолчанию.

Для того чтобы изменить объект или что-нибудь о нем узнать - пишем методы.

Связанный метод (bound method) - метод класса, который вызывается экземпляром.

Наследование классов позволяет создавать объекты, поведение которых похоже на другие объекты.

Наследование от стандартных классов:
>>> class Mylist(list):
...     def evven(self):
...         return len(self) % 2 == 0

Поддерживается множественное наследование - т.е. от несколькх классов сразу.

Посмотреть, является ли определеныый класс предком другого:
>>> issubclass(Mylist, object)
True
>>> issubclass(list, object)
True

Порядок наследования классов - слева-направо.
