Объектно-ориентированное программирование Python.

Как только используется ключевое слово class, Питон исполняет команду и создаёт объект. Инструкция

class Blog(object):
    
    def author_name(self, name = 'Ivan'):
        self.name = name
        return name

создаст в памяти объект с именем Blog. 


x = Blog()   # создание экземпляра класса

x.name()   # вызвать метод класса, вернет 'Ivan'

x.__setattr__('id', 1)   # создать аттрибут id и присвоить ему значение
x.id = 1    # тоже, более короткая запись

x.id   # получит значение аттрибута counter
